Principais formatações

Negrito / Destaque

Nesta frase temo um termo em negrito usando a tag <b>(não semântica)

Nesta frase, tenho um termo em destaque usando a tag <strong>(semântica)

Itálico / Ênfase

Nesta frase, temos um termo em Itálico usando a tag <i>

Nesta frase, usamos o termo <em>usado para dar ênfase em itálico

Atalhos usados para facilitar nossa vida Ctrl+Shift+P

Texto marcado

Podemos criar também um texto marcado usando a tag <mark> lembrando de usar o </mark> para fechar a tag

Aqui neste exemplo estou usando a tag mark

Texto grande e pequeno

Estamos criando um texto grande e um texto pequeno nesse parágrafo.

As tags usadas acima são: <big> = texto grande e <small> = texto pequeno

Podemos marcar um texto como excluído para indicar que ele deve ser lido, mas não considerado.

Também podemos marcar um texto como inserido para dar ênfase e indicar que ele foi adicionado depois

As tags usadas acima foram: <del> =Texto excluido e <ins> =Texto inserido

E a tag <u> deve ser usada para indicar que o texto possui uma anotação não textual, como uma correção ortográfica.
