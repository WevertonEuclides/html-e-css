Tags usadas acima: <a> =Vincula links externos a palavra

Tag usada para abrir o link externo em uma nova ABA: target="_blank" rel="external"

Exemplo: sem abrir em uma nova aba, <a href="https://www.youtube.com/@ForeverGOD1">Você pode acessar o meu canal do youtube</a>

Abrindo em uma nova ABA: <a href="https://www.youtube.com/@ForeverGOD1" target="_blank">Você pode acessar o meu canal do youtube</a>
